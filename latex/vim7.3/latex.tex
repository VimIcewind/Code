在汉字中间插入空格。比如下面的一行那个略微越出右边界的\VimDQuote{无}字\\

{
\fussy
\VimSepartor

但我却发现一处脚注里的文字在汉字之间插入了空格，排得比较匀称。把它COPY到
这里\VimMyFootnote{译注：Vim脚本中变量沿用了经典的计算机语言中变量的词法

4*. 如何在宏里实现替换？问题是这样的:\\
象这样的内嵌EMAIL地址\VimInlineURL{slimzhao@21cn.com}其中的
\VimDQuote{圈a}看起来是什么鬼样子？相信没几个人会欣赏，太容易
跟CopyRight的标志\copyright{}混淆了。

查了\VimLaTeX{}符号表后找到了marvosym包中一个\VimBS{}MVAt的符号，显示的效果是这样：\MVAt. 这是大家喜闻乐见的形式，这就有一个问题，我定义了一个宏来实现统一风格的MAIL超级链接:\\
\begin{VimVBshcmdNoAdjustSpace}[label=\VimLaTeX{}命令定义]
\newcommand{\VimMailURL}[1]
	{\textcolor{VimURLColor}{\href{mailto:#1}{#1}
}
\end{VimVBshcmdNoAdjustSpace}
\vspace{-1.3\parskip}

因为email地址是作为一个整体传给命令\verb|\VimMailURL|的，所以需要在宏定
义中想办法把参数\verb|#1|中出现的\verb|@|字符替换为\verb|\MVAt|.

目前的变通方法比较累：把宏修改成接受两个参数
\begin{VimVBshcmdNoAdjustSpace}[label=\VimLaTeX{}命令定义]
\newcommand{\VimMailURL}[2]{
	\texttt{<}\textcolor{VimURLColor}{\href{mailto:#1@#2}
	{#1{\footnotesize\MVAt}#2}}\texttt{>}}
\end{VimVBshcmdNoAdjustSpace}
\vspace{-1.3\parskip}

% \pdfbookmark[1]{view_10}{view_10}\hypertarget{view_10}{}\fbox{\textcolor{red}{Review}}\\
当然有了Vim可以很方便地替换所有的~\verb|\VimMailURL{xxx}{yyy.zzz}|~. 

\<rice\_maxwell@163.com>提供了更好的办法：
\begin{VimVBshcmdNoAdjustSpace}[label=Mail地址,xrightmargin=-3cm]
\def\<#1@#2>{\texttt{<}\href{mailto:#1@#2}{#1{\footnotesize\MVAt}#2}\texttt{>}} 
使用时可以这样：
\<abc@null.com>
\end{VimVBshcmdNoAdjustSpace}

5. 完美的中文字体\\

宋体字看着太淡。在6.3版的手册中汉字主体用的是文鼎公司的简报宋，好象没人
对此提出太大意见，我个人觉得比默认的宋体美观。没有找到更好的字体之前，就
